Split (csv) files into multiple parts with file cursors (smallest unit of split is a line), so that we can split dumping tasks, utilizing multiprocessing.

static split(path, num, jump=0, dist_random=False, single_threshold=1048576)

Split file into parts, returning a list of file pointers (that points into the file).

Parameters

  • path (str) – path to target file being split

  • num (int) – number of slices to split into

  • jump (int) – number of lines to jump (maybe because they are headers)

  • dist_random (int) – whether split files with random lines in each slice

  • single_threshold (int) – least file size so that we split it 文件小于该值不做切割
